# Homework 43

Цей проект був створений для практики з контекстом в REACT.

## Встановлення та запуск

Склонуйте репозиторій та виконайте команди:

```sh
git clone https://github.com/maxickdrobot/home-work-43
cd home-work-43
npm install
npm run dev
```

## Використані технології

-   React
-   React-router
-   TypeScript
-   Node.js
